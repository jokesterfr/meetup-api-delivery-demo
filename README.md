# meetup-api-delivery-demo

## Build and use

```
pnpm install
pnpm build
pnpm start
```

## Docker

```
pnpm build:docker
```

## Bootstrap log

```
npm i -g @nestjs/cli pnpm
nest new meetup-api-delivery-demo
cd ./meetup-api-delivery-demo
pnpm install
```
